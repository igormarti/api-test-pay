## Requirements

<ul>
  <li>MongoDB</li>
  <li>NodeJS 18.16.0(LTS) or above</li>
  <li>NPM 9.5.1 or above</li>
</ul>

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

The application will run on the following host: <a href="http:127.0.0.1:3000" >http:127.0.0.1:3000</a>
